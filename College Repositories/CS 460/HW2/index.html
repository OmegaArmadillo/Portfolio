Vector Calculator

The best calculator for vector calculation.

This calculator allows you to input vectors and calculate either the dot product or cross product for those vectors. To select 2D or 3D vectors slect from the Vector Dimension buttons. To calculate the Dot or Cross Product select form the Equation buttons. Input the x, y, and z values for each vector into the correct areas. Then press the calculate button to calculate the selected equaiton. Below the button the step-by-step calculations will be shown.
Dot Product: U*V=U1V1+U2V2+U3V3
Cross Product: UxV=(UyVz − UzVy),(UzVx − UxVz),( UxVy − UyVx)
